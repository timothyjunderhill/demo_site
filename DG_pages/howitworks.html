How Dash Galaxy Works

Language / Game Engine?

Dash Galaxy is written in the Swift programing language with the SpriteKit game engine.

Graphics / Models?

The models and backgrounds used in Dash Galaxy were all made with Affinity Photo

Game Modes?

The game mode mechanics are actually very simple. A new game will always start out in space mode, and when the player dies, the mode will also always return to space mode. When a player hits a portal the mode will switch to planet mode (when the current mode is space mode) or to space mode (when the current mode is planet mode). The game keeps track of the current mode at all times and this lets it know when to spawn certain objects.

World Types?

A World Type is randomly chosen by a simple process within the game. A number will be randomly generated and depending and the size of this number a world type will be chosen. Within this World Type several backgrounds, objects, and mechanics will be randomly selected following a similar process

Shields?

Shields are commonly found throughout the galaxy. When a player hits a shield a shield will generate around the player's rocket. This shield will protect the player from damage for 5 seconds before wearing off. A shield is automatically generated at the start of every game.
